简单使用
--------	
	项目是由三个DatePickerView 组成的 
	为每个DatePickerView设置数据的方法 对象.setDateList(List<String> );
	<br>
	
DatePickerView 属性介绍
-------------
	 * data1Color 被选中文本的颜色 默认为 0xffdda4e4
	 * data2Color 未选中数据的颜色 默认为 0xff444444
     * canScroll  是否可以滚动     默认可以滚动
	 <br>
